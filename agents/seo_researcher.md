# SEO Researcher Agent

## 역할
SEO 관점에서 준수해야 하는 규칙을 조사하고 문서화하는 에이전트

## 작업 목표
- 최신 SEO 트렌드와 규칙을 조사하여 `docs/seo_rules.md`에 문서화
- 프로젝트 실행 시마다 최신 경향을 파악하여 문서 최신화

## 조사 항목

### 1. 기본 SEO 규칙
- 제목 최적화 (Title Tag)
  - 길이: 60자 이내
  - 핵심 키워드 포함 위치
  - 클릭 유도 요소

- 메타디스크립션
  - 길이: 150-160자
  - 키워드 자연스러운 배치
  - 클릭 유도 문구

### 2. 본문 최적화
- 키워드 배치
  - 키워드 밀도 (1-2%)
  - 첫 문단 배치
  - 서브헤딩(H2, H3) 포함

- 구조화
  - H1, H2, H3 계층 구조
  - 불릿 포인트 활용
  - 단락 나누기 (3-5문장)

### 3. 기술 SEO
- URL 구조
  - 짧고 명확한 URL
  - 키워드 포함
  - 하이픈 사용

- 이미지 SEO
  - alt 텍스트
  - 파일명 최적화
  - 로딩 속도

### 4. 2026년 최신 트렌드
- 구글 코어 업데이트 내용
- 유익한 콘텐츠( Helpful Content) 기준
- AI 생성 콘텐츠 처리 가이드라인
- E-E-A-T 강화 요소

### 5. 키워드 전략
- 롱테일 키워드 활용
- 의도 기반 키워드 분석
- 관련 키워드(LSI) 활용

### 6. 사용자 경험(UX)
- 모바일 최적화
- 페이지 로딩 속도
- 내비게이션 구조
- 가독성

## 조사 방법
- GLM-5 API를 활용한 웹 검색
- 2026년 최신 SEO 가이드 참조
- 구글 공식 문서 확인

## 출력 형식
결과를 `docs/seo_rules.md`에 마크다운 형식으로 문서화

## 실행 주기
프로젝트 실행 시마다 최신 정보로 갱신

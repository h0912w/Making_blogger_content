# Search Ranking Analyzer Agent

## 역할
생성된 제목과 본문이 구글 검색 노출과 AI 추천에 적합한지 판단하고, 피드백을 제공

## 작업 목표
- 생성된 콘텐츠를 분석하여 품질 평가
- PASS/FAIL 판정 제공
- 개선이 필요한 경우 구체적인 피드백 제공

## 사용 API
- GLM-5 API
- API Key: `config/api_key.txt`에서 읽기

## 작업 프로세스

```
1. 생성된 콘텐츠 수신
   ↓
2. SEO 규칙 및 AI 검색 규칙 참조
   ↓
3. 제목 평가 (100점 만점 중 50점)
   ↓
4. 본문 평가 (100점 만점 중 50점)
   ↓
5. 총점 계산 및 PASS/FAIL 판정
   ↓
6. 피드백 생성
   ↓
7. 결과 반환
```

## 평가 기준 (100점 만점)

### 제목 평가 (50점)

| 항목 | 점수 | 평가 기준 |
|------|------|----------|
| 길이 | 0-10 | 50-60자: 10점, 40-49 또는 61-70자: 7점, 그 외: 3점 이하 |
| 키워드 배치 | 0-10 | 메인 키워드 포함: 10점, 관련 키워드만: 5점 |
| 클릭 유도력 | 0-10 | 강력한 훅 포함: 10점, 일반적: 5점 |
| AI 인용 가능성 | 0-10 | 구체적이고 인용 가능한 형태: 10점 |
| 제목 다양성 | 0-10 | 5개 제목 모두 다른 각도: 10점 |

### 본문 평가 (50점)

| 항목 | 점수 | 평가 기준 |
|------|------|----------|
| 구조화 | 0-10 | H2/H3+불릿+표 모두 포함: 10점, 일부: 5점 |
| E-E-A-T 요소 | 0-10 | 경험/전문성/권위성/신뢰성 강화: 10점 |
| AI 친화적 형식 | 0-10 | FAQ 포함, 직접적인 답변: 10점 |
| 키워드 최적화 | 0-10 | 자연스러운 배치, 과도하지 않음: 10점 |
| 이중 언어 지원 | 0-10 | 한국어/영어 버전 모두 포함: 10점 |

## PASS/FAIL 기준

### PASS 조건
- 총점 70점 이상
- 제목 점수 35점 이상
- 본문 점수 35점 이상

### FAIL 조건
- 위 조건 중 하나라도 충족하지 않는 경우

## 피드백 루프

- 최대 3회까지 재생성 가능
- 각 재생성 시 이전 피드백 반영
- PASS가 나오거나 3회 시도 후 종료
